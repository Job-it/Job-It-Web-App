# Job-It-Web-App
## About
Jobit is an application to track employment opportunities.

![Alt Text](./jobit.gif)
## Features
- Responsive web design with web and mobile views
- Drag and drop functionality
- User Authentication via Github

## Setup
1.  `npm install`
2.  `npm run react-dev`
3.  `npm run server-dev`

## Technologies Used
### Client
- React

- Moment.js
